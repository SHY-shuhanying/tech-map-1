# 技术关联分析地图

## 项目介绍
这是一个技术关联分析地图，用于展示不同技术领域之间的关联关系，包括洗护、洁面和护肤技术。

## 快速开始

### 本地预览
1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动本地服务器：
   ```bash
   npm run dev
   ```

3. 打开浏览器访问：`http://localhost:3000`

### 一键部署

#### 方法1：使用 Vercel 部署
1. 登录 Vercel 账号：https://vercel.com
2. 点击 "New Project"
3. 选择 "Import Git Repository" 或 "Drag and Drop"
4. 上传项目文件夹或连接 Git 仓库
5. 点击 "Deploy" 完成部署

#### 方法2：使用 Netlify 部署
1. 登录 Netlify 账号：https://netlify.com
2. 点击 "Add new site" -> "Deploy manually"
3. 拖拽项目文件夹到上传区域
4. 等待部署完成

#### 方法3：使用 GitHub Pages 部署
1. 创建 GitHub 仓库
2. 上传项目文件
3. 在仓库设置中启用 GitHub Pages
4. 选择 "main" 分支作为源

## 项目结构
```
├── index.html          # 主页面
├── clean_tech_map.html # 技术地图页面
├── package.json        # 项目配置文件
└── README.md           # 项目说明
```

## 技术栈
- HTML5
- Tailwind CSS
- D3.js
- Chart.js
- Font Awesome

## 功能特性
- 技术关联可视化地图
- 市场产品应用分析
- 未来技术发展方向
- 创新建议
- 爆品脑洞生成
- 文献与技术关联分析

## 部署注意事项
- 这是一个纯静态网站，无需后端服务
- 所有数据都内嵌在 HTML 文件中
- 可以部署到任何静态网站托管服务
- 支持自定义域名
